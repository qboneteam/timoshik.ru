title: "Перечисление значений переменной в цикле (Jekyll)"
date: 2020-04-27
---


В заголовке определяем значения переменных:
```
---
somevars: [1980 , 1979 , 1978]
---
```

Затем перечисляем их:
{% raw %}
``` 
{% if page.somevars -%}
    {% for somevar in page.somevars %}
        {{ somevar }}</br>
    {% endfor %}
{% endif %}
```
{% endraw %}
